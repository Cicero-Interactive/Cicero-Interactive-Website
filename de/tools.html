Ein simpler Media Player, basierend auf dem Windows Media Player.
Der "MediBurn Player" war ein Spaß-Projekt, das zum einen aus Langeweile entstand und zum anderen der Fortbildung meiner Programmierkenntnisse diente.
Funktionen: Kann (wie der Windows Media Player) die meisten Medienformate (solange sie nicht zu exotisch sind) abspielen, DVDs und Blu-Rays leider aber ausgenommen.
Nur in Deutsch verfügbar.

Made with Visual Basic (VB.NET)

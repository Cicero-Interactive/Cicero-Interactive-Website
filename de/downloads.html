This download is not the game but only a closed alpha scene. It is not a demo and is not a beta.
This is an alpha of a simple prototype scene, to show the work we are doing.
All advice and feedback is welcome.

Hardware Requirements
The final requirements of the product are not yet estimated with precision,
but we can assume that it will be enough for a mid range PC to run the game smoothly.
Currently, the alpha is not optimized as it will be in the final game and the frame rate is not 100% reliable.
Our goal is still to allow a gaming experience with no FPS drops maintaining these configurations:

Minimum configuration:
RAM 2GB
OS 32-bit Windows Vista or higher
DirectX 11 (Vista will require manual download and install of DirectX 11 if you don't already have it)
HDD Space (not yet estimable, but currently ~110MB)

Recommended configuration:
RAM 8GB
OS 64-bit Windows 7 or higher
DirectX 11
HDD Space (not yet estimable, but currently ~110MB)


Ingame basic keys:
Press the ESC key to open the pause menu.
Press the W key to fly up.
Press the A key to fly left.
Press the D key to fly right.
Press the S key to fly down.
Press the SPACE key to shoot.
Go to the settings under the point CONTROLS to see the default controls map.
